CODERvATION

"The impact you can have as a programmer is huge, as what you create can not just change your life, but lives of millions of others. Coding is probably the fastest way to change the world"

CoderVation is ACM style programming contest to be held on codechef. There will be 4-6 algorithmic real life problem which you have to solve using C/C++/Java. The problem will be of varying difficulty, which are made so considering beguiner and intermediate level of programmers.

Topics:

  1. Arrays
  2. strings
  3. searching
  4. sorting technique like insertion sort
  5. merge sort
  6. stach and queues
  7. recursive functions

 

Bug-De-Code

“Debugging is twice as hard as writing the code in the first place. Therefore, if you write the code as cleverly as possible, you are, by definition, not smart enough to debug it.” – Brian W. Kernighan

The key to success is failing at the right time. Failing is inevitable, but if you fail at the right time (e.g., while practicing), then you can learn by your mistakes, and do them less often in crucial moments like in a contest. In this event programs with bugs in them will be given to you and you have to debug them.

Bugs can be of types:

  1. Syntax error
  2. Semantic error
  3. Logical error
  4. Runtime error

C-Legacy

If you're a beginner - practice with your friends, read blogs of top programmers, solve easy challenges and improve with regular practice

This event is conducting aiming first year students in mind who only have learned or learning C language. This is all about C language and nothing else. In other events other programming language like c++/javav can be used but here only C.

Prequisites:

  1. C language
  2. Gcc compiler
  3. IDE like CodeBlock(optional)

 

Fast-Boot

We create the hardware and we create the software, the hardware makes maximum of software and software make maximun of hardware. ―Apple Inc.

This event is different from all other events. In this you have to first deassemble and then assemble a desktop and install an operating system on it in minimum amount of time, So no programming skill is required in this part.

Operating system:

  1. Windows Xp/7/8/10
  2. Linux ubuntu/fedora/Mint/kali/arch/other
